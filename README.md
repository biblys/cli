# biblys/cli

Biblys Command Line Interface

## Install

```shell
git clone
cd cli
npm install -g .
```

## Usage

### `deploy`

Deploy a site

```shell
biblys deploy [site] [version]
```

Deploy all sites

```shell
biblys deploy all [version]
```

### `version`

Display a site's current version

```shell
biblys version [site]
```

Display all sites current version

```shell
biblys version all
```
